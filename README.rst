
Quickstart
----------
    install python 2.7.13, 
    $ apt-get install python-dev build-essential

    $ git clone https://github.com/thinhlx1993/css-selector-backend
    
    $ cd css-selector-backend
    
    $ virtualenv env
    
    $ source env/bin/activate 
    
    $ pip install -r requirements.txt

    $ python run.py
